# Socratopia Anki 选择题模板转换工具使用说明

这个文件夹用于把 Socratopia 导出的 `.txt` 卡片文件转换成 Anki 可导入的选择题卡片。转换后的卡片可以在正面点击 A/B/C/D，并立即显示正确或错误。

## 文件说明

- `prettify-unified-mcq-stable-template.apkg`  
  Anki 模板包。第一次使用前先导入它，用来创建笔记类型 `Prettify Unified MCQ Stable`。

- `socratopia_txt_to_anki_gui.py`  
  转换工具本体。需要电脑已安装 Python。

- `打开_Socratopia转换工具.bat`  
  Windows 双击启动入口。一般直接双击这个文件即可打开转换工具。

## 第一次使用

1. 打开 Anki。
2. 点击 `文件` -> `导入`。
3. 选择 `prettify-unified-mcq-stable-template.apkg`。
4. 导入后，Anki 中会多出一个笔记类型：`Prettify Unified MCQ Stable`。
5. 导入模板包后出现的测试卡可以删除。

## 转换 txt 文件

1. 双击 `打开_Socratopia转换工具.bat`。
2. 点击 `选择文件`，选择 Socratopia 导出的原始 `.txt` 文件。
3. 输出路径会自动生成，通常不需要修改。
4. 牌组名默认是 `Socratopia`，可以按需修改。
5. 笔记类型保持 `Prettify Unified MCQ Stable`。
6. 点击 `转换`。
7. 转换成功后，会得到一个 `.tsv` 文件。

## 导入转换后的 TSV

1. 打开 Anki。
2. 点击 `文件` -> `导入`。
3. 选择刚转换出来的 `.tsv` 文件。
4. 确认导入设置：
   - 笔记类型：`Prettify Unified MCQ Stable`
   - 牌组：`Socratopia` 或你设置的牌组名
   - 分隔符：Tab
   - 允许 HTML：开启
   - 第 11 列作为 Tags
5. 点击导入。

## 已经复习过旧卡怎么办

如果你之前已经导入过同一批卡，并且已经复习了一段时间，不要删除旧卡。

这个工具生成的 TSV 会把原始 `Front` 保留在第 1 列，用来让 Anki 识别已有笔记。重新导入时，如果 Anki 提示更新已有笔记，选择更新即可。这样通常可以保留原来的复习记录。

导入前建议先在 Anki 中创建备份：`文件` -> `创建备份`。

## 注意事项

- 不要随意修改原始题干和选项文字。Anki 主要靠第 1 列判断是不是同一张卡，题干大改后可能会被识别成新卡。
- 如果转换工具提示某些行无法解析，通常是该行不符合 A/B/C/D 四选项格式。
- 如果双击 `.bat` 没反应，请确认电脑已经安装 Python，并且 Python 已加入系统 PATH。
- 这个工具不需要安装第三方 Python 库。

## 推荐流程

以后每次拿到新的 Socratopia `.txt` 文件，都按这个顺序：

1. 用本工具把 `.txt` 转成 `.tsv`。
2. 在 Anki 中导入 `.tsv`。
3. 使用统一模板 `Prettify Unified MCQ Stable` 复习。
